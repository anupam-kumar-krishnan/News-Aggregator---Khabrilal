## _Features_ ✨🚀
📌 A News Aggregator implemented with <b>_Django Framework_</b><br>
📌 Various <b>_Categories_</b> of News are presented on a <b>_Single Page_</b><br>
📌 It is implemented using News API by <b>_Google_</b><br>
